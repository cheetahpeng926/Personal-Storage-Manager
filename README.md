# 📦 Personal Storage Manager

**个人物品仓库管理系统**

管理衣物、鞋子、电子设备、汽车模型等个人物品，支持图片预览与 Excel 导出

</div>

---

## ✨ 功能特性

- **九大分类管理**：短袖、长袖、短裤、长裤、袜子、鞋子、汽车模型、驾驶经历、电子设备
- **增删改查**：弹出表单录入，必填校验，自动格式化日期与比例字段
- **模糊搜索**：按品牌、型号等文本字段实时过滤
- **列排序**：点击表头在升序 ▲、降序 ▼、默认 ↕ 之间循环切换
- **图片管理**：每条记录支持添加 / 更换 / 删除图片，右侧面板实时预览，点击可用系统程序打开原图
- **Excel 导出**：一键导出全部分类到 Excel（每个分类一个独立 Sheet）
- **窗口记忆**：自动保存并恢复上次窗口位置
- **Apple 风格简洁 UI**：圆角按钮、自定义椭圆滚动条、浅色主题

## 🛠️ 技术栈

| 组件       | 技术                                 |
| ---------- | ------------------------------------ |
| 语言       | Python 3.8+                          |

## 🚀 快速开始

### 1. 环境要求

- Python 3.8+
- 可选： [MiSans](https://hyperos.mi.com/font/download) 字体（缺失时自动回退到黑体）

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

如需 Excel 导出功能，额外安装：

```bash
pip install openpyxl
```

### 3. 启动

**Windows 免安装版**

前往 [Releases](https://github.com/cheetahpeng926/Personal-Storage-Manager/releases) 下载 Personal Storage Manager.exe 放在空文件夹中，直接双击即可使用

**或下载源码后执行**

```bash
python main.py
```

首次启动会在同目录自动创建 `data.db` 数据库及九张数据表，**无需任何数据库配置。**

## 💡 使用提示

- **日期格式**：支持 `YYYY.MM.DD`，粘贴 8 位数字自动加点（`20240315` → `2024.03.15`）
- **比例字段**：中文冒号自动转英文（`1：18` → `1:18`）
- **图片格式**：支持 JPG / PNG / GIF / BMP / WEBP
- **列排序**：点击表头循环切换 → 升序 ▲ → 降序 ▼ → 默认
- **导出**：点击左下角「导出」按钮，生成带格式的 Excel 文件
- **双击编辑**：双击表格行可直接打开编辑表单

## ⚠️ 注意事项

- 数据存储在同目录的 `data.db` 文件中，注意备份
- 图片按分类自动存储在 `Pictures/` 子目录下
- 删除记录时会同步删除关联的图片文件
- 默认仅支持本地使用，如需远程访问请自行改造
